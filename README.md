# Food101Classifier 🍲
Food 101 Dataset Classifier

🎯 Goal: Classify 101 food.

Dataset: [Food 101](https://pytorch.org/vision/main/generated/torchvision.datasets.Food101.html)
Most techniques are taken from https://www.learnpytorch.io/

Includes:
- Conv2D model
- Transfer learning
- ViT implementation
